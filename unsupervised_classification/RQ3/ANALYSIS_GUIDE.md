# RQ3 可视化分析指南

## 核心概念速查

### 1. Bias 的含义
- **Bias** = 余弦相似度 cos_sim(文档向量, 道德轴向量)
- **正值** → 文本语义偏向**正向动机**（关爱、公平、忠诚、安全感、文化认同）
- **负值** → 文本语义偏向**负向动机**（伤害、欺骗、背叛、威胁、腐蚀）→ **攻击性动机**

### 2. Z-score 的含义（图A, B, C, D）
- **z_score** = (bias - 该轴全样本均值) / 标准差
- **用途**：消除"语言整体基线差异"，突出"该语言内部的相对重点"
  - 例如：中文整体 Bias 较负，但用 z-score 后可判断"在中文语言内部，哪些轴最突出"
- **含义**：z < -1.5 → 该语言/话题在该轴特别负向（突出攻击性）

### 3. 7个道德轴的含义

| 轴名 | 正极 (+) | 负极 (-) | 解读 |
|------|---------|---------|------|
| **Harm** | 关爱、慈悲 | 伤害、虐待 | 个人伤害；最基础的道德直觉 |
| **Fairness** | 公平、互惠 | 欺骗、剥削 | 规则与不公正；制度层面 |
| **Loyalty** | 爱国、忠诚、凝聚 | 背叛、渗透、分裂 | 群体认同；内外有别 |
| **Authority** | 权威、秩序、法律 | 颠覆、叛乱、无序 | 层级与纪律；制度合法性 |
| **Sanctity** | 神圣、纯洁、健康 | 堕落、病理化、异化 | 宗教/文化圣洁性；纯净性 |
| **Realistic Threat** | 安全、稳定、资源稳定 | 资源抢夺、政治控制 | 群际竞争的实际利益 |
| **Symbolic Threat** | 文化凝聚、传统继承 | 文化入侵、价值观腐蚀 | 身份认同的文化威胁 |

## PCA 坐标的定性含义（图D, G2, H2）

### PC1：攻击性程度 / 负向梯度
- **含义**：文本在所有7个轴上的总体负向程度
- **计算逻辑**：PCA 提取方差最大的方向 → 第一主成分通常反映了"最大的群体差异"
- **正方向** → 更多正向轴（关爱、公平、忠诚、权威、圣洁、安全、文化凝聚）→ **建设性、合理的道德关注**
- **负方向** → 更多负向轴（伤害、欺骗、背叛、颠覆、堕落、威胁）→ **攻击性、仇恨的道德动机**

**如何读 PC1**：
- 图中靠左（PC1 < -1）= 该话题/语言群体中充满了攻击性动机
- 图中靠右（PC1 > +1）= 该话题/语言群体强调建设性、正向价值

### PC2：动机类型对比轴
- **含义**：区分不同类型的道德动机
- **常见的 PC2 分解**：
  - 上方（PC2 > 0）：可能强调 **宏观群体轴**（Loyalty, Authority, Symbolic Threat）
  - 下方（PC2 < 0）：可能强调 **微观伤害轴**（Harm, Fairness, Sanctity）

**如何读 PC2**：
- 图中靠上 = 强调"群体身份、权威、威胁"的攻击动机
- 图中靠下 = 强调"个人伤害、公平、纯洁"的攻击动机

## 【图解】三个分析维度对比

### 📊 维度一：By Language（图 G1, G2）
**问题**：三种语言的整体道德动机签名是什么？

#### 图 G1：语言×轴 热力图
- **行**：三种语言（英中日）
- **列**：7个道德轴  
- **颜色**：原始 Bias 均值
- **读法**：
  - 找最红的格子 = 该语言最有攻击性的轴
  - 看某语言的"红蓝条纹模式" = 该语言的道德关注点重排序
  
**例子**：
- 如果英文的 "Sanctity" 轴是红色（-0.1 负值），日文是中性（0.0），中文更负（-0.2）
  - → 中文圣洁/纯洁轴的攻击性最强 = 中文更多用"纯洁被污染"的叙事来攻击宗教

#### 图 G2：三语言在 PCA 空间的位置
- **点位置**：该语言的中心位置（所有话题语言内部的平均 PC1/PC2）
- **椭圆**：该语言的方差范围（±1σ）
- **读法**：
  - 点离开原点（0,0）距离 = 该语言的"非主流"程度
  - 椭圆大小 = 该语言内部的多样性

**例子**：
- EN 在 (PC1=+0.3, PC2=-0.1)，中文在 (PC1=-0.5, PC2=+0.2)
  - → 英文更"正向 + 微观伤害驱动"，中文更"负向 + 宏观群体驱动"

### 📊 维度二：By Topic（图 H1, H2）
**问题**：不同话题的道德关注点排序是什么？攻击性最强的讨论集中在哪些话题？

#### 图 H1：话题×轴 热力图
- **行**：话题（按总体负向程度从上到下排序）
- **列**：7个道德轴
- **读法**：
  - 从上到下 = 从最负向到最正向的话题群落
  - 看每行的"颜色条纹" = 该话题的道德关注点独特性

**例子**：
- Topic 25 (Social Division and Racial Antagonism) 全是深红色 = 这个话题充满各种攻击性动机
- Topic 14 (Public Prayer and Religious Freedom) 只有某两个轴深红 = 这个话题的攻击性集中在特定维度

#### 图 H2：话题在 PCA 空间的位置
- **气泡位置**：该话题的中心位置（所有 (lang, topic) 单元的平均 PC1/PC2）
- **气泡大小**：该话题的文档数
- **气泡颜色**：PC1 值（红=负向/攻击，蓝=正向/建设）
- **读法**：
  - 左下象限（PC1 < 0, PC2 < 0）= 攻击性强 + 强调个人伤害  → 最激烈的个人仇恨
  - 左上象限（PC1 < 0, PC2 > 0）= 攻击性强 + 强调群体威胁 → 最激烈的身份冲突
  - 大气泡在坏位置 = 高传播的有害讨论

### 📊 维度三：By Topic-Language（图 A, B, C, D）
**问题**：每个 (语言, 话题) 配对的独特道德签名是什么？

#### 图 A：Topic-Language×轴 热力图（主图）
- **行**：每一行是一个 (语言, 话题) 配对，e.g. "EN · T25 · Social Division"
- **列**：7个道德轴
- **颜色**：该子群体在该轴的**语言内部 z-score**
- **最重要的特点**：用 z-score 而非原始 Bias
  - **为什么**：原始 Bias 有"语言人工制品"（比如中文 LLM 生成的向量可能有天然偏移）
  - **z-score** = 在该语言内部的相对地位
  
- **读法**：
  - 看某行最深的负红色格子 = 这个话题-语言配对在该轴最突出
  - 比较相同话题在不同语言的行 = 同一话题在三种语言的道德重排序

**例子**：  
- Topic 31 (Antisemitism) 在英文的 "Harm" 轴最红（z = -2.1），但在中文只是浅红（z = -0.8）
  - → 同样的反犹主义话题，在英文语境中更强调"个人伤害"，在中文可能更强调其他维度

#### 图 D：Topic-Language 动机空间（PCA biplot）
- **点**：每个 (lang, topic) 单元
- **点颜色**：语言（蓝/橙/绿）
- **箭头**：7个道德轴的方向和强度
- **读法**：
  - 某点靠近某个箭头 = 这个(lang, topic)单元强烈沿着该轴的负方向
  - 比较不同颜色的点的分群 = 不同语言的话题有不同的空间分布

## 补充：数据统计（表格）

### 表格 1：ANOVA 显著性检验（ANOVA_CSV）
- **F_stat**, **p_value**：三语言在该轴是否差异显著
- **eta_squared (η²)**：效应量大小（0-1）
  - η² > 0.14 = 大效应
  - η² 0.06-0.14 = 中效应
  - η² < 0.06 = 小效应
- **Tukey HSD p-values**：三语言两两比较
  - 哪两种语言差异最大？最小？

**读法**：如果 Harm 轴的 p=0.001, η²=0.18 → 三语言在强调"伤害"上有大幅差异

### 表格 2：Language-Axis 汇总（CSV_SUMMARY）
- 每种语言每个轴的：均值、标准差、分布形态

## 常见问题

**Q1: 为什么要用 z-score，不直接用原始 Bias？**  
A: 原始 Bias 可能包含"向量语言工程伪制品"（embedding 随语言不同有基线偏移）。z-score 消除语言内部基线，突出相对重点。

**Q2: PC1 和 PC2 是不是一定就是"负向"和"类型"？**  
A: 不一定。要看你的数据实际的主成分方向。建议：
- 看 loading 向量：各轴对 PC1/PC2 的贡献
- 看点的分布：点如何在图上排列

**Q3: 图 H1 中话题为什么要"按总体负向程度排序"？**  
A: 为了快速看出"最危险的讨论在哪里"。排序后，上面的话题是最激烈的仇恨泉源。

**Q4: 怎样判断三语言的"差异是否显著"？**  
A: 
1. 看 ANOVA 的 p-value（p < 0.05 = 显著）
2. 看 η² 的大小（η² > 0.14 = 大效应）
3. 看 Tukey HSD 的 p-values （指出哪两种语言差异最大）
